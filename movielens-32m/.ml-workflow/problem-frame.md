# Problem Frame

**Target:** Predict unseen movies a user would watch, given their watch history. Ranking/retrieval task on implicit feedback.
**Error priority:** Symmetric — missing good recommendations and surfacing irrelevant ones are equally costly.
**Constraints:** Must be highly performant but not too compute-heavy to train.
**Data:** `data/ratings.csv`, `data/movies.csv`, `data/tags.csv`, `data/links.csv` (MovieLens 32M)
